## 🧑🏻‍💻 HTTP 메시지와 구조
---

### ✅ Request 메시지
> 클라이언트에서 서버로의 메시지로, 헤더(메서드 + URI + 프로토콜 버전), 공백라인, 바디로 구성되어있다.

<center>
    <img src="/img/네트워크/5-1-1.png" width="350" height="400" alt="Request 메시지" />
</center>

### ✅ Response 메시지
> 서버에서 클라이언트로의 메시지로, 헤더(프로토콜 버전 + 상태코드 + 상태코드 설명), 공백라인, 바디로 구성되어있다.
